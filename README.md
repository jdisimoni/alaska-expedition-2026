# Alaska Expedition 2026

Companion site for Jay's July 16–23, 2026 Virgin Voyages Alaska cruise.

## Upload
Upload every file in this folder to the root of the `alaska-expedition-2026` repository.

## GitHub Pages
Settings → Pages → Deploy from a branch → `main` → `/ (root)`.

Expected URL:
https://jdisimoni.github.io/alaska-expedition-2026/

## Install on iPhone
Open the live site in Safari → Share → Add to Home Screen.
