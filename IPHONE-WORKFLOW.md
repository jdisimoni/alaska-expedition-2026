# iPhone Daily Update Workflow

## Send to ChatGPT

- Destination and date
- 3–5 favorite photos
- Three highlights
- Favorite meal
- Wildlife sightings
- Funniest or most unexpected moment
- Approximate photo and video counts
- Anything you want coworkers to know

## ChatGPT prepares

- Polished journal entry
- Photo captions
- Photo of the Day
- Updated route progress
- Updated wildlife tracker
- Updated photo/video/mileage counters
- Updated site package

## Publish from iPhone

1. Download the updated ZIP.
2. Open it in Files and extract it.
3. Open the GitHub repository in Safari.
4. Upload the replacement files.
5. Commit the update.
6. Wait for the Pages deployment.
7. Use the site's Share button.

The site uses versioned CSS and JavaScript URLs and removes the previous service-worker caches automatically.
